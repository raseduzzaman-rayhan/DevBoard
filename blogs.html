Quetion-1: What are the different ways to select an element in the DOM

There are several ways to select an element in the DOM such as getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll. Among them, querySelector and querySelectorAll are the most flexible as they support CSS-style selectors.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns or sets the HTML content of an element, including tags. innerText returns or sets the visible text, respecting styles like display: none. textContent returns or sets all text inside the element, including hidden text, but without HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is added to a parent element to handle events from its child elements. It works by taking advantage of event bubbling in the DOM.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behavior in the DOM where an event starts from the target (child) element and bubbles up to its parent elements. This allows parent elements to handle events triggered by their children.
